# server

## install dependencies

`npm install`
or
`yarn`

## serve with hot reload at localhost:4000

`npm start`
or
`yarn start`

# client

## install dependencies

`yarn install`
or
`npm install`

## serve with hot reload at localhost:3000

`yarn start`
or
`npm start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
